User Persona

About Project

* This project was about creating an user persona for Bhutanese individuals seeking manual labor or

blue-collar work opportunities in Australia.

To create this user persona the things that I had to do are;

1/ Research and Gather Data

2/Segment my Audience

3/Identify Goals and Needs

4/Created Persona Profiles

5/Design and Development

Key Takeaways

* User-Centered Design

* Audience Understanding

* Design Guidance
